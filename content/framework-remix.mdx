## 使用付費服務的原因 & 費用

### 使用付費服務 fly.io

想要使用 `MDX`，但是 Remix 目前把 `.mdx` 檔案當做是頁面的功能還非常陽春，例如：無法取得路徑名稱。連基本的 SEO 設定都無法滿足。因此，以 Remix 做的網站（部落格），有許多是 `mdx-bundler`，這也是 Remix 開發者做的套件。

## Markdown 外部連結

Markdown 的超連結語法：

```markdown
[Astro 首頁](https://astro.build)
```

會產生基本的超連結：
```html
<a href="https://astro.build">Astro 首頁</a>
```

但是，想要以新開分頁的方式開啟，而現在外部連結都強調安全性，應該要：
```html
<a href="https://astro.build" target="_blank" rel="noopener">Astro 首頁</a>
```
