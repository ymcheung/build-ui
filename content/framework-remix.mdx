## Markdown 外部連結

Markdown 的超連結語法：

```markdown
[Astro 首頁](https://astro.build)
```

會產生基本的超連結：
```html
<a href="https://astro.build">Astro 首頁</a>
```

但是，想要以新開分頁的方式開啟，而現在外部連結都強調安全性，應該要：
```html
<a href="https://astro.build" target="_blank" rel="noopener">Astro 首頁</a>
```
